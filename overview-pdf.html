Advanced Email Processing & Document Management

Our intelligent email processing system has been redesigned with professional NEC4 document templates and streamlined workflows. The system automatically processes incoming emails and creates appropriate contract records based on document type.

Professional NEC4 Document Templates

  • Equipment Hire Requests - Complete hire documentation with NEC4 Clause 70.1 compliance, including operator requirements, delivery details, and commercial terms
  • Request for Information (RFI) - Structured technical queries following NEC4 Clause 17.1 for ambiguities and inconsistencies, with programme impact assessment
  • Compensation Event Notifications - Formal CE documentation under NEC4 Clause 60.1, including physical conditions, time/cost impact, and supporting evidence requirements
  • Early Warning Notices - Risk management notifications per NEC4 Clause 16.1, with mitigation strategies and meeting requests
  • Professional Dropdown Selector - Intuitive interface for selecting appropriate document templates with clause references and category information
  • Authentic Contract References - Each template includes proper project references, NEC4 contract terminology, and industry-standard formatting
  • Automatic Record Creation - Intelligent classification and routing to appropriate contract management modules

Enhanced RFI Management System

Our Request for Information (RFI) management module integrates seamlessly with the email processing system to streamline communication, track requests, and ensure timely responses. The system now includes:

Key RFI Management Features

  • Intuitive Kanban Interface - View RFIs by status (Open, In Progress, Responded, Closed) with clear visual indicators
  • Contractual Reply Period Tracking - Automatically calculate and display the contractual response period for each RFI
  • GPS MACS Code Integration - Associate RFIs with appropriate GPS MACS codes for proper project accounting
  • Comprehensive RFI Details - View all information related to an RFI including descriptions, comments, and response history
  • Performance Metrics - Color-coded indicators for on-time vs. late responses with detailed analytics
  • Compensation Event Integration - Seamless connection between RFIs and related compensation events
  • Email Template Integration - Direct integration with professional NEC4 RFI templates for consistent formatting
